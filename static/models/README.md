# 3Dアバターモデル

- `avatar_real.vrm` — リアルモード(3D)用のVRMモデル
  - 出典: [pixiv/three-vrm](https://github.com/pixiv/three-vrm) のサンプルモデル (VRM1_Constraint_Twist_Sample)
  - ライセンス: [VRM Public License 1.0](https://vrm.dev/licenses/1.0/)、アバター利用許可は「everyone」
  - 差し替える場合は同名で上書きするか、`static/app.js` の `VRM_MODEL_URL` を変更する
    (blink / aa / happy などのVRM標準表情プリセットを持つモデルを推奨)
